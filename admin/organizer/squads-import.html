Команды этапа

Выбираешь этап → загружаешь CSV → данные появляются ниже по категориям.
Формат: матрица с ; как у твоего Excel.

Список команд

После импорта: тренер пустой, рейтинг = 3. Тут же можно править команду/город, рейтинг и тренера (по имени), удалять и добавлять вручную.
      Подсветка строки = есть несохранённые изменения. Bulk сохраняет только такие строки.
      Ниже можно выбрать связанные команды.

      Связанные команды / общий лимит нагрузки

      Используй это, если один и тот же состав играет в двух возрастах. Например: Команда 2013 + Команда 2014. Тогда генератор будет считать их одной физической командой и сможет ограничивать общую нагрузку по дню.
      Правило действует только в рамках выбранного этапа и должно ссылаться на команды этого этапа.

      Добавить команду

      Добавляется в текущий выбранный этап и активную категорию (вкладку).

      Добавить правило общей нагрузки

      Удерживай Ctrl / Cmd для выбора нескольких команд.
